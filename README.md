# MovieDB App

<p>React MovieDB is a movie search engine made with ReactJS and powered by The Movie Database API.  </p>
<h4>Build with</h4>
<ul>
  <li>ReactJs</li>
</ul>
<h4>Tools</h4>
<ul>
    <li>ReactJS</li>
    <li>Sass</li>
    <li>Axios</li>
    
</ul>
<h4>To get local copy up and running follow these simple steps:</h4>
<ul>
 <li>clone repo with git clone command</li>
 <li>create a .env file on root and open</li> 
 <li>add a variable REACT_APP_API_KEY and assign a Api key</li> 
 <li><b>npm install</b></li>
 <li><b>npm run start</b></li>
</ul>
